# Book Shelf

### タスク管理はNotionにて

[Notion](https://www.notion.so/Book-Shelf-9c391d7842844114b3e75eb1654a27a1?pvs=4)  

## 要件定義

- ユーザーは書籍のタイトル、著者、出版社、ISBNコード、感想などを登録できる。
- 登録された書籍は一覧表示され、ユーザーは閲覧できる。
- ChatGPTを利用した自動要約機能を実装し、書籍の要約を表示できる。
- Google Books APIを利用した全文検索機能を実装し、書籍の検索をサポートする。
- 書籍情報の取得機能を実装し、書籍の詳細情報を表示できる。

## 最小限の機能 (MVP)

- ユーザー登録とログイン機能
- ユーザーが書籍情報を追加、閲覧、編集できる機能
- Google Books APIとの連携による書籍検索、取得機能
- ChatGPTを使用した要約生成機能の実装
- 簡単なUI設計による使いやすさ

| 種類 | 技術 |
| --- | --- |
| フロントエンド | TypeScript |
| バックエンド | TypeScript, Node.js |
| データベース | MongoDB |
